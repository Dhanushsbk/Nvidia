# Deep Learning & Computer Vision Foundations

This repository contains a collection of Jupyter Notebooks and resources tracking a progressive journey through Deep Learning. It covers everything from basic artificial neural network (ANN) operations to fully functional Computer Vision models utilizing Convolutional Neural Networks (CNNs).

## 📂 Repository Structure

* 🧠 `neural.ipynb`: A foundational notebook showcasing the raw mathematics behind neural connections, forward propagation, and visual mapping.
* 👟 `fashionNeural.ipynb`: An implementation of a Multi-Layer Perceptron (MLP) built with TensorFlow/Keras to classify clothing types using the Fashion MNIST dataset.
* 🏔️ `intelImage.ipynb`: An end-to-end Convolutional Neural Network (CNN) computer vision project for classifying natural scene landscapes into 6 distinct categories.
* 📊 `intel image output.jpeg`: A sample visualization output showing the loss and accuracy training graphs generated by the Intel Image model.

---

## 🚀 Projects Overview

### 1. Neural Network Foundations (`neural.ipynb`)
This notebook covers the fundamental math required to understand how a neural network computes outputs before frameworks like TensorFlow abstract it away.
* **Concepts Implemented:** Vector dot products, inputs, weight matrices, and bias addition.
* **Math Visualized:** $\text{Output} = \mathbf{X} \cdot \mathbf{W} + b$
* **Visualizations:** Generates a clean connection plot using `matplotlib` to visually map input layers, a hidden layer, and output nodes.

### 2. Fashion MNIST Classification (`fashionNeural.ipynb`)
Moving from raw math to practical framework training, this notebook trains a Deep Neural Network (DNN) to handle low-resolution grayscale imagery.
* **Dataset:** **Fashion MNIST** (70,000 $28 \times 28$ grayscale images across 10 clothing categories).
* **Architecture:** Sequential model utilizing Flatten, Dense (ReLU activated), and an output Softmax layer.
* **Inference:** Includes an evaluation script that plots a test grid displaying the real product image alongside its predicted classification category (e.g., *T-shirt, Trouser, Pullover, Dress, Coat, Sandal, Shirt, Sneaker, Bag, Ankle Boot*).

### 3. Intel Image Scene Classification (`intelImage.ipynb`)
An advanced exploration of spatial feature extraction via Convolutional Neural Networks (CNNs) handling complex, full-color ($150 \times 150$) image sets.
* **Dataset:** **Intel Image Classification** (Contains roughly 25,000 real-world terrain images).
* **Target Classes:** `buildings`, `forest`, `glacier`, `mountain`, `sea`, and `street`.
* **Model Pipeline:**
  * Image array normalization and batch scaling.
  * CNN feature extraction through repeating blocks of 2D Convolution (`Conv2D`) and downsampling (`MaxPooling2D`).
  * Dropout regularization to mitigate overfitting.
  * Model serialization saving the trained pipeline weights directly to an `intel_image_classifier.h5` file.

## 📈 Model Performance Tracking

The repository tracks learning efficiencies using training history validation splits. The model generates twin graphs to evaluate training progression over epochs:

* **Model Accuracy:** Measures training accuracy vs. cross-validation accuracy to check for generalizing reliability.
* **Model Loss:** Uses categorical cross-entropy loss tracking to verify the error drops continuously.

*(Refer to `intel image output.jpeg` inside the repository to see an example visualization of these training curves).*
