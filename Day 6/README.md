# AI Projects Collection

## Overview

This repository contains two Artificial Intelligence and Machine Learning projects developed using Python and Jupyter Notebook.

### Projects Included

1. Transformer Model Benchmarking (BERT vs GPT-2)
2. AI Text-to-Vector Image Generator using Gemini

# Project 1: Transformer Model Benchmarking (BERT vs GPT-2)

## Description

This project benchmarks and compares the performance of two popular Transformer-based models:

* BERT (bert-base-uncased)
* GPT-2

The notebook evaluates both models using a common text input and measures performance metrics such as latency, memory usage, token throughput, and context window size.

## Technologies Used

* Python
* PyTorch
* Hugging Face Transformers
* psutil
* Jupyter Notebook

## Features

* Automatic CPU/GPU detection
* Memory consumption analysis
* Inference latency measurement
* Token throughput calculation
* Comparative performance reporting

## Metrics Evaluated

* Latency (seconds)
* Memory Usage (MB)
* Tokens Processed
* Tokens per Second
* Context Window Size

## Learning Outcomes

* Understanding Transformer architectures
* Performance benchmarking techniques
* Resource utilization analysis
* Comparison of encoder and decoder models

# Project 2: AI Text-to-Vector Image Generator using Gemini

## Description

This project uses Google's Gemini API to generate vector artwork from natural language prompts.

The AI creates SVG and HTML-based illustrations that are rendered directly inside a Jupyter Notebook. Users can provide creative prompts and receive AI-generated vector graphics.

## Technologies Used

* Python
* Google Gemini API
* SVG
* HTML
* Jupyter Notebook
* IPython Display

## Features

* AI-powered vector graphic generation
* SVG rendering inside notebooks
* Prompt-based artwork creation
* Fast content generation using Gemini
* Creative design automation

## Workflow

1. User enters a text prompt.
2. Gemini generates SVG/HTML artwork.
3. Generated code is cleaned.
4. Artwork is rendered directly inside the notebook.

## Applications

* AI-generated illustrations
* Graphic design prototyping
* Educational demonstrations
* Creative AI projects
* SVG artwork generation

## Future Enhancements

* Interactive prompt interface
* SVG export functionality
* Multiple art styles
* Batch image generation
* Custom color themes
