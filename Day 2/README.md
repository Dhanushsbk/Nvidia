# Titanic Survival Prediction using Machine Learning

## Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning. A Logistic Regression model is trained on the Titanic dataset using passenger information such as class, age, and fare.

## Features

* Data preprocessing and cleaning
* Handling missing values
* Train-test data splitting
* Logistic Regression model training
* Survival prediction
* Model evaluation using Accuracy Score and Confusion Matrix
* Data visualization

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Dataset

The project uses the Titanic dataset (`train.csv`) containing passenger details such as:

* Passenger Class (Pclass)
* Age
* Fare
* Survival Status (Target Variable)

## Project Structure

```text
titanic.ipynb
train.csv
README.md
```

## Installation

Install the required libraries:

```bash
pip install pandas scikit-learn matplotlib
```

## How It Works

1. Load the Titanic dataset.
2. Select important features:

   * Pclass
   * Age
   * Fare
3. Handle missing values in Age.
4. Split data into training and testing sets.
5. Train a Logistic Regression model.
6. Predict passenger survival.
7. Evaluate model performance using accuracy and confusion matrix.

## Example Output

```text
Accuracy: 82%
```

## Future Enhancements

* Use additional features such as Sex and Embarked.
* Compare multiple classification algorithms.
* Hyperparameter tuning.
* Deploy as a web application using Flask or Streamlit.
